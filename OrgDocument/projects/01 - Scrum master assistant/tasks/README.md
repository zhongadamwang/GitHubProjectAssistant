# Task Management - Scrum Master Assistant

## Overview
This directory contains all tasks related to the Scrum Master Assistant project. Tasks are managed using GitHub issue format for consistency and traceability.

## Task Tracking
See [`task-tracking.md`](task-tracking.md) for current task status and progress overview.

## Creating New Tasks
Use the template in [`task-template.md`](task-template.md) to create new tasks. Each task should be saved as a separate markdown file with a descriptive filename.

## Task Organization
- **Active Tasks**: Tasks currently being worked on
- **Backlog**: Tasks planned for future sprints
- **Completed**: Finished tasks (archived)
- **Blocked**: Tasks waiting on dependencies or decisions

## Naming Convention
Tasks should be named using the format: `task-[category]-[brief-description].md`

Examples:
- `task-requirements-user-stories.md`  
- `task-analysis-technical-architecture.md`
- `task-development-core-framework.md`

## Status Updates
Update [`task-tracking.md`](task-tracking.md) regularly to reflect current task status and progress.
